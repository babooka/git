  • git config - параметры git
  • git init - создание локального репозитория
  • git status - список файлов в репе
  • git add - добавление репа в индекс
  • git commit - фиксация, зименений, комментирование
  • git log - история изменений в репе
  • git checkout - переключение на другую ветку
  • git branch - показывает все локальные ветки
  • git push - заливка файлов в удаленный реп
  • git pull - получить последние изменения с удаленного сервера
  • git merge - слияние в одну ветку
  • git clone - создает новый каталог и делает git init

    • .gitignore файл - позволяет исключить передачу файлов или папок в git репозиторий, понимает регулярные выражения

  • git revert - отмена операции в удаленном репозитории (возврат изменений)
  • git reset - отмена индекса файла, но сохранение его содержимого
  • git rebase - перенос коммита с ветки на ветку
  • git cherry-pick - перенос некоторых коммитов и объединение в 1
  • git stash - отложить изменения и вернуть ветку в исходное состояние
